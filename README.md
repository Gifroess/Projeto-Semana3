# Lista de Tarefas

Projeto desenvolvido durante a terceira semana da capacitação da área de Projetos Desktop da Asimov Jr.

O objetivo desta atividade foi aprender os conceitos fundamentais de TypeScript e aplicá-los no desenvolvimento de uma lista de tarefas interativa. O projeto permitiu praticar tipagem estática, manipulação do DOM e organização de código, além de implementar funcionalidades que auxiliam no gerenciamento de atividades do dia a dia.

## Tecnologias utilizadas

- TypeScript
- HTML5
- CSS3

## Funcionalidades

- Adição de novas tarefas;
- Exclusão de tarefas;
- Marcação de prioridade;
- Definição de prazo para conclusão;
- Identificação de tarefas atrasadas;
- Marcação de tarefas concluídas;
- Estatísticas de tarefas concluídas;
- Filtro de visualização por:
  - Todas as tarefas;
  - Tarefas pendentes;
  - Tarefas concluídas.
